## Coding 2 Final Project

For your final project, you will develop a Python-based command-line interface (CLI). Your program must prompt the user to enter data, then your function should be able to perform an operation, a conditional, and a loop and return new data. 

### Project Requirements
Your program must have the following components
- your program must have at least one user defined function
- your progrm must take in atleast 1 input
- your program function needs to use atleast 1 parameter
- your program must perform at least 1 operation (arithmetic, comparision, etc..)
- your program should have a conditional (if/else), and a a loop (for or while)
- your program must print out some data.
